For a variable being declared in code, the length of the name should be at least 4 characters.

Noncompliant Code Example

class MyClass {
  void doSomething() {
    int n = 3; // Noncompliant
    String str; // Noncompliant
  }
}

Compliant Solution

class MyClass {
  int doSomething(int number) { // Compliant
    int numberPlusTwo = number + 2; // Compliant
    return numberPlusTwo;
  }
}
